# Project Aether: Vertical AI Platform
# Status: PROPRIETARY SYSTEM | Strategy: Data Moat | Aesthetic: Molecular Coalescence

## 1. The Vertical AI Thesis (The "Moat")
We are a **Data Engine** that correlates abstract "Vibes" with molecular "Ground Truth."
- **Layer 1 (The Consultant):** User Input -> Vector Search -> Molecular Reveal.
- **Layer 2 (The Truth Loop):** Blind Date Reveal -> User Verification -> Proprietary Dataset.

## 2. Tech Stack
- **Frontend:** Flutter (Riverpod, GoRouter).
- **Physics:** `flutter_animate` (Required for particle/coalescence effects).
- **Typography:**
  - Headers: **Cinzel Decorative** (Bold, Spaced 1.5) - "Quinten" Aesthetic.
  - Body: **Cormorant Garamond** (Italic) - "Literary" Aesthetic.

## 3. Feature Specifications

### Feature A: The "Scent-Sona" Input
- **Style:** "Invisible Input" (No borders).
- **GenUI:** Background gradient morphs in real-time (Rain -> Teal, Oud -> Amber).
- **Action:** "CONSULT THE ORACLE" button fades in.

### Feature B: The "Molecular Coalescence" (Results)
- **Concept:** Gas -> Liquid -> Solid.
- **Visual Implementation (The "Aether Orb"):**
  - **Start:** A large, glowing `RadialGradient` orb (using the Vibe Color, e.g., Teal). Shape: Circle. Blur: High.
  - **Animation:** The Orb shrinks and morphs into a Rectangular aspect ratio.
  - **End:** The colored gradient cross-fades into the Product Image.
- **Rule:** NEVER show a black placeholder. Always use the generative vibe color.

### Feature C: The "Blind Date" Reveal (Data Capture)
- **Concept:** The Unboxing Ritual.
- **UI:** A frosted glass container hiding the bottle.
- **Interaction:** "Press and Hold" button triggers a "Flash" animation.
- **Data Capture:** On reveal, the system **MUST** log the `DATA MOAT` event to the console.

## 4. Design System
- **Palette:** Alabaster (`#FAF9F6`) Background, Charcoal (`#1A1A1A`) Text, Metallic Gold (`#D4AF37`) Accents.
- **Navigation:** Drawer Icon must be **Charcoal** or **Gold** (High Contrast).

## 5. API Contract
- **POST /api/v1/analyze-vibe:** Returns list of scents.
- **POST /api/v1/blind-date-reveal:** Logs the ground truth data (User ID + Scent ID + Verified Match).
